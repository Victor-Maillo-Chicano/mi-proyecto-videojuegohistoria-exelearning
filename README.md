# mi-proyecto-videojuegohistoria-exelearning
Proyecto en grupo destinado a 1º de la ESO en el que el alumnado debe crear la base o beta de un videojuego de temática histórica una vez realizada una recopilación de datos históricos, seleccionado personajes históricos o inventado tramas o personajes ficticios, todo ello haciendo uso de herramientas digitales.
